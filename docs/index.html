Drawing I는 어떤 서비스인가요?

Drawing I는 아동의 HTP 그림검사를 디지털로 구현하여
그리는 과정을 분석하고, 상담사의 보고서 작성을 도와주는
AI 기반 그림심리검사 보조 EMR 서비스입니다.

환자 등록하기

상담사는 웹 Admin 페이지에서 피검사자 정보를 등록할 수 있습니다. 이름, 성별, 주민등록번호, 주소, 가족력, 의뢰사유 등의 정보를 입력하면 해당 아동의 검사 권한이 자동으로 생성됩니다.

HTP 그림검사 진행

아동은 태블릿에서 검사를 진행합니다. 집 → 나무 →
사람 → 사람(다른 성별) 순서로 그림을 그리고 그림을 완료하면 이어서 그림에 대한 질문이 주어집니다.
아이는 질문에 대해 직접 답을 입력하거나 음성 입력
기능을 사용해 말로 대답할 수 있습니다.

검사 과정 분석

① 타임라인 기반 검사 흐름 확인

타임라인 뷰어를 통해 전체 검사 영상이 시간 흐름에 따라 구성됩니다. 상담사는 원하는 시점에 메모를 추가하거나
시간을 클릭하여 해당 시점의 영상으로 바로 이동할 수 있습니다. 이는 검사 흐름의 맥락 파악과 상담 기록 작성에 효과적입니다.

검사 과정 분석

② 행동 이벤트 기반 리컨스트럭션

시스템은 검사 과정 중 의미 있는 행동을 자동으로 탐지합니다. 선 굵기 변화, 지우고 다시 그린 영역,
빠르게 그린 부위, 반복된 그림 등은 클릭 시 해당 시점의
획(stroke) 정보로 재현됩니다. 이는 아동의 행동에 담긴 심리적 의미를 직관적으로 분석할 수 있게 합니다.

리포트 초안 자동 생성

검사 결과는 GPT 기반으로 요약되며, 아이가 직접
입력한 질문 응답과 상담사가 남긴 메모도 자동 반영
됩니다. 또한, 최종 그림에서 객체(집, 나무, 사람 등)가
검출되어 HTP 보고서 양식에 항목별로 입력됩니다.
결과 리포트는 상담사가 자유롭게 가공 후 병원 EMR 시스템에 저장하여 활용할 수 있습니다.
