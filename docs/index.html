👶 프로젝트 소개

저희는 미술 심리 상담 보조 서비스 Drawing I 를 개발했습니다. 저희 서비스를 통해 상담사가 검사 품질과 업무 효율성면에서 큰 향상을 얻게 하는 것이 저희의 가장 큰 목표입니다.

🧑‍💻 팀원 소개

  • 김설(팀장) : AI
  • 오성우 : AI
  • 고다연 : Backend
  • 안세준 : Backend
  • 이지수 : Frontend
  • 정성엽 : Frontend

📱 주요 기능

  • 그림 검사 과정을 사용하기 편하게 가공해주는 기능
  • 분석 결과를 영상과 함께 타임라인 형태로 시각화 시켜서 상담사에게 제공
  • 타임라인은 하이퍼링크같은 인터랙션 요소처럼 동작하여, 상담사가 즉시 보고싶은 시점으로 이동
  • 검사 보고서 작성 자동화 지원
